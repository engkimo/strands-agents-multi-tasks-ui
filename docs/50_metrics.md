# 指標 / KPI

## ビジネス/KPI
- 週間有効ユーザー（WAU）、比較実行率、再実行率、セッションあたり成功実行数。

## プロダクトヘルス
- 実行成功率、平均/95p レイテンシ、ノード失敗率、リトライ成功率。
- UI 体感: 初回描画 < 2s、更新レイテンシ < 200ms。

## 品質（将来の評価連携）
- タスク別の品質スコア（テスト合格、ルーブリック採点、要件充足率）。
- コスト最適化: 1 タスクあたりコスト/品質の効率指標。

## コスト/運用
- 実行あたりの API コスト（推定）、ストレージ使用量、テレメトリ送信量。

## 計測実装
- OpenTelemetry を第一級で採用、Span 属性にツール/ノード/コスト/トークンを付与。
- SQLite/ファイルで最小実装、将来外部可視化基盤へ出力も可能に。
